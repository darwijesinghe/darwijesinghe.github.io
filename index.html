Projects

Assigna

Task Manager Web Application

This web application was built to showcase basic CRUD operations using ASP.NET Core. It includes functionality to add, update, and delete tasks, send reminders, and filter tasks by category. User authentication and registration, including local and external (Google) logins, are managed through the .NET Core Identity Framework. Data is handled using the .NET Core Entity Framework with SQL Server as the database. Assigna is designed for small teams to easily manage their project tasks.

Assigna API

Task Manager REST API

This RESTful API was created to demonstrate basic CRUD operations using ASP.NET Core. It supports adding, updating, and deleting tasks, sending reminders, and filtering tasks by categories. The API is secured with JWT tokens and authorization is enforced based on user roles. Data management is handled with .NET Core Entity Framework and SQL Server. AssignaApi can be utilized as a backend solution for building a small task management application.

Battleship API

The Classic Battleship Game Logic

BattleshipAPI is a RESTful API that implements the classic Battleship game logic. The API allows players to place ships on a grid and fire shots to try to sink their opponent's ships. The project is built using .NET 6 and follows a client-server (three-tier) architecture pattern. This includes a console app, and it serves as the client communicating with the API.

Online Store

Ecommerce Store Web Application

This application was developed for CFC Healthcare, a leading pharmacy chain in Sri Lanka. It features Webxpay integration for payments, a shopping cart, user profile management, and support for multiple product variations. The solution was built using the ASP.NET MVC design pattern, SQL Server, HTML, CSS, and JavaScript, providing a full-scale online store experience.
