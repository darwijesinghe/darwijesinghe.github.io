Projects

Assigna

Task Manager Web Application

This web application was developed to demonstrate basic CRUD operations using ASP.NET Core. Includes Add / Update / Delete tasks, Send reminders, and filter by Categories. Local / External (Google) login and registration are handled by using .NET core Identity Framework. Data managed with .NET Core Entity Framework and SQL Server. Assigna can use for a small team to simply manage their project tasks.

Assigna API

Task Manager REST API

This RESTful API was developed to demonstrate basic CRUD operations using ASP.NET Core REST API. Includes Add / Update / Delete tasks, Send reminders, and filter by Categories. Secure with JWT Tokens and Authorized by user roles. Data managed with .NET Core Entity Framework and SQL Server. AssignaApi can use it as a backend project to develop another small task management application.

Assigna Console

Task Manager Console Application

This console application was developed to demonstrate basic CRUD operations using .NET Core. Includes Add / Update / Delete tasks, Send reminders, and filter by Categories. This full application was developed by consuming Assigna WEB API.

CFC Online

Ecommerce Store Web Application

This application was developed for the client CFC Healthcare which has a well-established pharmacy chain in Sri Lanka. Includes Webxpay payments, shopping cart, profile manager, and different variations of products. This full-scale online store solution was developed by using ASP.NET MVC design pattern, SQL Server, HTML, CSS, and JavaScript.
